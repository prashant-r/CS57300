# CS573-DataMining-HW4-FangdaLi
### Python 2.7
### Command Line Usage:
```
usage: hw4.py [-h] trainingDataFilename testDataFilename modelIdx

Train and test decision tree based classifiers.

positional arguments:
  trainingDataFilename  Name of the training data csv file
  testDataFilename      Name of the test data csv file
  modelIdx              Model to use: 1 for Decision Tree, 2 for Bagged
                        Decision Trees, 3 for Random Forest, 4 for Boosted
                        Decision Trees, 5 for SVM.
```
